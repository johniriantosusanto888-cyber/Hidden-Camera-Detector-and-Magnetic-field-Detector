# Hidden Camera Detector APK
APK for testing Hidden Camera Detector and Magnetic Field Detector app

The app uses your phone’s camera and built-in sensors to detect unusual reflections and magnetic field disturbances that may indicate hidden surveillance equipment. Key Features: • Camera lens reflection detection • Magnetic field (EMF) detection for nearby electronics • Real-time scanning using your phone camera

How to Install
1. Download the APK from releases on GitHub.
2. Enable Unknown Sources in your phone settings.
3. Install the APK and open the app.

Notes
1. This app is lightweight and does not require internet.
2. All data is stored locally on your device.
